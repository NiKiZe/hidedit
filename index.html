About

The "Human Interface Device" class of peripheral devices includes USB and Bluetooth mice, keyboards, joysticks, game pads, and everything else which interacts with humans.

According to the specification, every HID peripheral provides its own "HID Report Descriptor", which tells the host (e.g. your PC or Smartphone) what the device's features are and how to communicate with it and make use of these features.

This report descriptor is written in a binary format which is tedious to edit by hand, so a tool like HIDEdit is very useful when you need to write such a descriptor, or edit it.

Why

This project aims to provide hardware developers with a usable, intuitive and lightweight online editor for HID report descriptors. Before HIDEdit, the standard tool for this job was the HID Descriptor tool from usb.org, which is very old, somewhat buggy, and only runs on Windows.

Several alternative tools exist, but none of them is free (neither gratis nor libre). HIDEdit is both free to use and is free software (GPL). The core of HIDEdit is written in pure Javascript and the UI is DHTML. This allows HIDEdit to work on all modern browsers regardless of the specific platform.
